# Retro Tool

Next.js와 Firebase Firestore를 이용해 만드는 회고 보드 초안입니다.

## 포함된 기능

- 프로젝트 비밀번호 입력
- 기본 섹션 3개: `잘한점`, `부족한점`, `노력한점`
- 섹션 이름 인라인 수정
- 카드 추가 / 수정 / 삭제
- Firestore `boards/main` 문서에 저장

## 시작 방법

1. 의존성 설치
2. Firebase 프로젝트 `woogobee`에서 Firestore 데이터베이스 생성
3. 아래 규칙 또는 인증 정책 설정
4. `npm run dev`
